# 🖨️ မြန်မာ အလှူခံကတ် ပရင့်ထုတ်စနစ် (Myanmar Automated Donation Card Printer)

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

**ND Donation Card Printer** သည် ကျောင်းထိုင်ဘုန်းကြီးကျောင်းများ၊ သီလရှင်ကျောင်းများနှင့် ပရဟိတအဖွဲ့အစည်းများအတွက် အလှူရှင်များ၏ အချက်အလက်များ (အမည်၊ လှူဒါန်းငွေ၊ လိပ်စာ၊ ရက်စွဲ) ကို အသင့်ရိုက်နှိပ်ထားသော အလှူခံကတ် (Physical Pre-printed Cards) များပေါ်ရှိ အစက်ချမျဉ်းများပေါ်သို့ မီလီမီတာ (Millimeter) အတိအကျဖြင့် လှပသေသပ်စွာ ပရင့်ထုတ်ပေးနိုင်သော Professional Desktop Application တစ်ခု ဖြစ်ပါသည်။

---

## 🌟 အဓိက လုပ်ဆောင်ချက်များ (Key Features)

### ၁။ 📊 Excel ဖိုင် စီမံခန့်ခွဲမှုနှင့် အလိုအလျောက် ရွေးချယ်ပေးခြင်း
- **Excel ဖိုင် ရွေးချယ်နိုင်ခြင်း**: `DonorsData.xlsx` သာမကဘဲ မိမိစိတ်ကြိုက် အမည်ဖြင့် သိမ်းဆည်းထားသော အလှူရှင်စာရင်း Excel ဖိုင်များ (`May_Donations.xlsx`, `Special_Fund.xlsx`) ကို **"📂 Excel ဖိုင်ရွေးမည် (Select File)"** ခလုတ်ဖြင့် လွယ်ကူစွာ ဖွင့်လှစ်နိုင်ပါသည်။ ဖွင့်ထားသော ဖိုင်အမည်ကို အလိုအလျောက် မှတ်သားထားမည် ဖြစ်ပါသည်။
- **Smart Last-Entry Default Selection**: အလှူရှင်အသစ် တစ်ဦးချင်းစီ ထပ်မံဖြည့်စွက်သည့်အခါ မြန်ဆန်စွာ ပရင့်ထုတ်နိုင်ရန်၊ ဇယားကို ဖွင့်လိုက်သည်နှင့် **နောက်ဆုံးအလှူရှင် (Last entry) ကိုသာ အလိုအလျောက် ရွေးချယ် (☑ ထုတ်မည်)** ထားပေးမည်ဖြစ်ပြီး ဇယားကိုလည်း အောက်ဆုံးသို့ အလိုအလျောက် ရွှေ့ပေးထားမည် (Auto-scroll to bottom) ဖြစ်ပါသည်။
- **မြန်မာဂဏန်းနှင့် ငွေကြေးပုံစံ (Myanmar Formatting)**: Western numbers (`0123456789`) များကို မြန်မာဂဏန်း (`၀၁၂၃၄၅၆၇၈၉`) များသို့ အလိုအလျောက် ပြောင်းလဲပေးပြီး၊ `200000` ကို `၂၀၀,၀၀၀  - ကျပ်` အဖြစ် ကော်မာ (Comma) များဖြင့် သေသပ်လှပစွာ ပြင်ဆင်ပေးပါသည်။

### ၂။ 🇲🇲 ၁၀၀% ပြီးပြည့်စုံသော မြန်မာစာ OpenType ပုံစံ (Flawless Typography)
- **Chromium OpenType Shaping Engine**: ယခင် သာမန် PDF စနစ်များတွင် တွေ့ရလေ့ရှိသော `ရွှေ` သည် `ရေွှ` ဖြစ်နေခြင်း၊ `မြို့နယ်` တွင် အပေါ်အောက် အသတ်များ ထပ်နေခြင်း ပြဿနာများကို အပြီးတိုင် ဖြေရှင်းနိုင်ရန်၊ Windows ၏ မူလ **DirectWrite / HarfBuzz OpenType Text Shaping Engine (Microsoft Edge / Google Chrome Headless Engine)** ကို တိုက်ရိုက် အသုံးပြုထားပါသည်။
- ထို့ကြောင့် မည်သို့ပင် ရှုပ်ထွေးသော မြန်မာစာလုံးပေါင်းဖြစ်စေ Microsoft Word နှင့် Excel တို့တွင် မြင်ရသည့်အတိုင်း **၁၀၀% အတိအကျ ပြီးပြည့်စုံစွာ မှန်ကန်နေမည်** ဖြစ်ပါသည်။
- **အလိုအလျောက် စာကြောင်းခွဲပေးခြင်း (Intelligent Word Wrapping)**: အလှူရှင်အမည်နှင့် လိပ်စာ ရှည်လျားပါက သတ်မှတ်ထားသော အကျယ်ပေါ်မူတည်၍ စာကြောင်းများကို အလိုအလျောက် သေသပ်စွာ ခွဲထုတ်ပေးပါသည်။

### ၃။ 🎨 ဖောင့်နှင့် နေရာ ချိန်ညှိမှုစနစ် (Precision Alignment & Custom Fonts)
- **စက်တွင်း ဖောင့်များ (System Fonts)**: စက်ထဲရှိ `C:\Windows\Fonts` တွင်ရှိသော `Pyidaungsu`, `Myanmar Text` စသည်တို့အပြင် **"📂 ဖိုင်ရွေးမည် (Browse)"** ခလုတ်ကို နှိပ်၍ မိမိကွန်ပျူတာထဲရှိ မည်သည့် `.ttf` သို့မဟုတ် `.otf` ဖောင့်ကိုမဆို ထည့်သွင်းအသုံးပြုနိုင်ပါသည်။ (`@font-face` ဖြင့် PDF ထဲသို့ အတိအကျ ထည့်သွင်းပေးပါသည်)။
- **စာသား နေရာချထားမှု (Alignment & Line Spacing)**: စာသားများကို သက်ဆိုင်ရာ အစက်ချမျဉ်းများ၏ **အလယ်တည့်တည့် (Center on line)** တွင် အတိအကျ ရိုက်နှိပ်ပေးမည်ဖြစ်ပြီး၊ စာကြောင်းအကွာအဝေး (Line Spacing) ကိုလည်း pt ဖြင့် စိတ်ကြိုက် ချိန်ညှိနိုင်ပါသည်။
- **စာသားအရောင် (Font Colors)**: အပြာရင့် (Royal Blue `#0066cc`)၊ အပြာတောက်တောက် (Electric Blue)၊ အနက်ရောင်၊ ရွှေအိုရောင် စသည့် အရောင်များကို စိတ်ကြိုက် ရွေးချယ်နိုင်ပါသည်။

### ၄။ ⚙️ နေရာ အတိအကျ ချိန်ညှိခြင်းနှင့် Dynamic Versioning (Advanced Calibration)
- **Baseline Alignment**: စာသားများ၏ အောက်ခြေ (Baseline) နှင့် ကတ်ပေါ်ရှိ အစက်ချမျဉ်းတို့ တစ်ထပ်တည်းကျစေရန် Y ဝင်ရိုးကို -18 pt အထက်သို့ အတိအကျ ဆွဲတင် ချိန်ညှိပေးထားပါသည်။
- **Expanded Sliders Range**: ပရင်တာအမျိုးအစားပေါ် မူတည်၍ စာရွက်ဆွဲယူမှု ကွာခြားပါက စာသားများကို လိုအပ်သလို ရွှေ့နိုင်ရန်အတွက် X နှင့် Y ဘားတံများ (Sliders) ကို **-100 pt မှ +100 pt အထိ** ကျယ်ပြန့်စွာ တိုး/လျှော့ ချိန်ညှိနိုင်ပါသည်။
- **Dynamic PDF Versioning (`_v12345.pdf`)**: Windows တွင် PDF ဖိုင် ဖွင့်ထားဆဲဖြစ်ပါက ဖိုင်ကို Lock လုပ်ထားတတ်သော ပြဿနာကို ကျော်လွှားနိုင်ရန်အတွက် ပရင့်ထုတ်မည်ကို နှိပ်လိုက်တိုင်း PDF ဖိုင်အမည်နောက်တွင် Version နံပါတ်အသစ်များ အလိုအလျောက် တပ်ပေးထားသောကြောင့် Lock ဖြစ်ခြင်း လုံးဝမရှိတော့ဘဲ နေရာချိန်ညှိမှု (Calibration) အားလုံး ချက်ချင်း မြင်တွေ့ရမည် ဖြစ်ပါသည်။
- **အလိုအလျောက် မှတ်သားပေးထားခြင်း (Persistent Memory)**: မိမိရွေးချယ်ထားသော ဆက်တင်များ အားလုံးကို `printer_config.json` တွင် အလိုအလျောက် မှတ်သားပေးထားပါသည်။

### ၅။ 🖨️ ပရင့်မုဒ် (၂) မျိုး (Dual Output Modes)
1. **ကတ်အလွတ်ပေါ် ထပ်ရိုက်မည့်မုဒ် (Print Overlay Mode)**: အသင့်ရိုက်နှိပ်ထားသော Physical ကတ်အလွတ်များအတွက် စာသားသီးသန့်ကိုသာ ကတ်ပေါ်ရှိ နေရာအလွတ်များပေါ်သို့ အတိအကျ ရိုက်နှိပ်ပေးမည် ဖြစ်ပါသည်။
2. **ဒီဇိုင်းနောက်ခံပါ ပရင့်ထုတ်မည့်မုဒ် (Full Certificate Mode)**: ဒီဂျစ်တယ်ကတ်အဖြစ် အသုံးပြုရန် သို့မဟုတ် ပရင့်မထုတ်မီ စာသားနေရာ ချိန်ညှိမှု မှန်ကန်ခြင်း ရှိ/မရှိ စစ်ဆေးရန်အတွက် နောက်ခံဒီဇိုင်းပုံ (`blank_card_template.jpg`) ပေါ်တွင် တိုက်ရိုက် ပေါင်းစပ်ရေးဆွဲပေးမည် ဖြစ်ပါသည်။

## 🚀 အသုံးပြုနည်း (How to Use)

၁။ ဖိုင်တွဲထဲရှိ **`Start_DonationCard_App.bat`** ကို နှစ်ချက်နှိပ် (Double-click) ၍ ဖွင့်ပါ။ (အက်ပ်ပွင့်လာသည်နှင့် Excel မှ နောက်ဆုံး အလှူရှင်ကိုသာ အလိုအလျောက် ရွေးထားပေးမည်ဖြစ်ပါသည်)။
၂။ လိုအပ်ပါက **"📂 Excel ဖိုင်ရွေးမည် (Select File)"** ကို နှိပ်၍ မိမိ၏ အလှူရှင်စာရင်း Excel ဖိုင်ကို ရွေးချယ်ပါ။
၃။ ပရင့်မုဒ် (Print Overlay သို့မဟုတ် Full Certificate) ကို ရွေးချယ်ပါ။
၄။ ဖောင့်အမျိုးအစား၊ အရောင်နှင့် လိုအပ်ပါက X/Y နေရာများကို ချိန်ညှိပါ။
၅။ **"🖨️ ပရင့်ထုတ်မည့် PDF ဖန်တီးမည် (Generate Print PDF)"** ခလုတ်ကို နှိပ်လိုက်သည်နှင့် ချက်ချင်းလှပသေသပ်သော PDF ဖိုင် ထွက်ပေါ်လာမည် ဖြစ်ပါသည်။

## 🛠️ နည်းပညာ တည်ဆောက်မှု (Technical Stack & Requirements)

- **Language**: Python 3.8+
- **GUI Framework**: Tkinter (with custom modern TTK Clam styling)
- **Data Processing**: Pandas (`pandas`, `openpyxl`)
- **PDF Generation Engine**: Headless Chromium / Microsoft Edge DirectWrite OpenType HarfBuzz Engine (`subprocess` API)
- **File Manipulation**: PyMuPDF (`pymupdf`), Pillow (`Pillow`)

### Requirements Installation
```bash
pip install pandas openpyxl pymupdf pillow
```
